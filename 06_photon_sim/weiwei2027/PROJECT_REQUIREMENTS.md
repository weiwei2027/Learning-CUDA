## 领域知识简介

X射线成像利用不同组织对X射线衰减程度的差异来形成图像。当X射线穿过头部时，光子与组织（骨骼、软组织、空气）相互作用，部分被吸收，剩余光子到达探测器形成投影。线性衰减系数 μ 描述材料对X射线的吸收能力，单位 cm⁻¹，依赖于光子能量和组织类型。

每个光子的自由程（两次相互作用之间的距离）由随机数和 μ 决定；若自由程小于到组织边界的距离，则光子被吸收，否则穿越边界进入不同组织区域。最终统计到达探测器各像素的光子数，即可模拟出投影图像。该计算天然并行，每个光子独立，适合 GPU 加速。

难点在于几何边界处理（颅骨、软组织分层、可能存在的异物）和高效统计。本项目中，头部简化为多层平板（皮肤、颅骨、脑组织），可包含一个球形异物（如血块或金属碎片），以模拟病变。

## 输入文件定义

### 1. 几何模型文件（文本格式）

描述头部简化为多层平板，可选包含一个球形异物。每行定义一个区域。

**格式：**
```
type name thickness x_start y_start z_start material
```

**字段说明：**
- `type`：`layer`（平板层）或 `sphere`（球形异物）
- `name`：区域名称（仅用于识别）
- `thickness`：对于 layer，表示该层的厚度（cm）；对于 sphere，表示半径（cm）
- `x_start, y_start, z_start`：区域起始坐标。对于 layer，表示该层起始点（z坐标）；对于 sphere，表示球心坐标
- `material`：材料名称，与物理参数文件中的材料对应

**示例（头部三层加一个血块）：**
```
layer skin 0.2 0.0 0.0 0.0 tissue_skin
layer skull 0.8 0.0 0.0 0.2 tissue_bone
layer brain 16.0 0.0 0.0 1.0 tissue_brain
sphere hematoma 0.5 5.0 5.0 5.0 tissue_blood  # ✅ 已实现
```

> **实现状态**: 平板层 (`layer`) ✅ 已完全支持；球体 (`sphere`) ✅ 已完全支持

**解释：**
- 皮肤层厚度 0.2 cm，从 z=0 开始
- 颅骨层厚度 0.8 cm，从 z=0.2 开始
- 脑组织层厚度 16 cm，从 z=1.0 开始
- 在脑内 (5.0,5.0,5.0) 处有一个半径 0.5 cm 的血肿块（模拟颅内出血）
- 探测器位于最后一层之后，如 z=17.2 cm

### 2. 物理参数文件（CSV格式）

每种材料在不同X射线能量下的线性衰减系数 μ（cm⁻¹）。

**格式：**
```
material,energy_keV,mu
```

**字段说明：**
- `material`：材料名称，如 `tissue_skin`、`tissue_bone`、`tissue_brain`、`tissue_blood`
- `energy_keV`：光子能量（keV）
- `mu`：线性衰减系数（cm⁻¹）

**示例：**
```
tissue_skin,50,0.2
tissue_skin,100,0.15
tissue_bone,50,0.5
tissue_bone,100,0.3
tissue_brain,50,0.18
tissue_brain,100,0.12
tissue_blood,50,0.22
tissue_blood,100,0.16
```

> **注：** 若光子能量不在表中，可通过线性插值获得对应μ值。✅ 已实现

### 3. 源参数文件（文本格式）

定义X射线源的特性。格式为键值对：

**示例：**
```yaml
source_type = point                    # ✅ point 或 parallel
position = 0.0 0.0 -1.0                # 源位置 (x,y,z) cm
direction = 0.0 0.0 1.0                # 主方向单位向量
energy = 50.0                          # 单能光子能量 (keV)
num_photons = 10000000                 # 模拟光子总数
detector_position = 0.0 0.0 17.5       # 探测器平面位置（z坐标）
detector_size = 20.0 20.0              # 探测器尺寸 (宽,高) cm
detector_pixels = 1024 1024            # 像素网格 (宽,高)
```

**字段说明：**
- `source_type`：`point` 表示点源（锥形束），`parallel` 表示平行束
- `position`：点源的三维坐标
- `direction`：主方向单位向量
- `energy`：光子能量
- `num_photons`：要模拟的光子总数
- `detector_position`：探测器平面所在的z坐标（假设探测器垂直于z轴）
- `detector_size`：探测器在x和y方向的宽度（cm）
- `detector_pixels`：像素网格数

## 输出文件定义
